# 00 · 项目总览

## 项目名
**CanvasDeadlines** — macOS 菜单栏小程序

## 目标
让 UCLA Extension 学生 Ewan 能在 Mac 菜单栏一键查看所有 Canvas 课程「最近 10 个未提交的作业/测验」，按截止时间由近到远排序。

## 范围（本期 v1.0）

**包含**：
- 菜单栏图标 + 下拉面板
- 自动从 Canvas API 拉取所有活跃课程的 assignments + quizzes
- 倒计时显示（"还剩 N 天" / "今天截止" / "已逾期 N 天"）
- 两种视图：全部混合 / 按学科分组
- 标记提交（手动）+ 自动检测已提交项并隐藏
- 跳过项（隐藏，可在设置中恢复）
- 过期项标红
- 首次启动 Token 配置引导

**不包含**（明确放后期）：
- 系统通知 / 桌面提醒
- 多账户支持
- 提交内容编辑
- 离线缓存
- 自动启动登录项管理
